<h2>Introduction:</h2> </br>
We created a full-stack web application called Smart Fix that is an on-demand, solutions-based application connecting users to service providers with their day to day repairs and maintenance needs. 

<h2>Motivation:</h2></br>
We recognize the need for a platform that can enable users to easily connect with reliable service providers. We wanted to create a platform that enables us to add certified service providers to our database, and their performance backed by user reviews. This will help create more jobs for service providers while providing an easy-to-use platform for the users.

<h2>How Does Smart Fix Work:</h2></br>
In order to use the application, users and service providers have to sign in using their designated login platforms. Users are able to post for a new job and service providers are able to give a price quote for the respective job as well as accept and decline job offers.

<h2>Technologies Used:</h2></br>
Node.js, Express.js, Handlebars.js, MySQL, Sequelize and ORM.
